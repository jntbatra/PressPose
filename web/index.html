CrossSense

Attentive cross-modal sensor pairing — given two time-series sensor streams, decide whether they came from the same person. Learned directly on raw signals.

GitHub repo → Docs

The idea

Two sensor streams, one question: same person? CrossSense encodes each modality with a small Transformer over time, then lets the two streams attend to each other before a pairing head — learning directly on raw signals, with no hand-crafted features, image rendering, or ImageNet backbone (unlike the original VGG16+LSTM baseline it grew from).

Results

Validated on real multi-subject data via a different cross-modal pairing task on the public UCI HAR dataset (accelerometer ↔ gyroscope, same-person?). 30 subjects, scored subject-disjoint — test people are never seen in training.

0.811Accuracy
0.810Macro-F1
0.872ROC-AUC

⚠️ This is a generalisation demo on UCI HAR (accel↔gyro) — not the plantar-pressure / 2D-pose benchmark, which is pending the full released dataset. The synthetic curve below is a CI sanity check only, not a benchmark.
